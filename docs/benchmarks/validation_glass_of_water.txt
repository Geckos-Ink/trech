TRECH benchmark reference: Glass-of-Water Optical Validation
=============================================================

This file is a permanent, diff-friendly snapshot of the inverse
classical-optics validation against TRECH's Geant4 optical photon
transport.  Re-running scripts/validate_glass_of_water.py
regenerates this file in place; commits that move any number
trigger a visible diff so engine regressions / improvements are
caught in code review.

Scenario:           examples/experiments/validation_glass_of_water.js
Validator:          scripts/validate_glass_of_water.py
Run metadata:       4000 trajectories, beam 2.2500 eV (~ 551 nm)
